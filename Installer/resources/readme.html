Loop Breaker — About

Loop Breaker is a creative audio instrument plug-in by GlowMachineAudio. It captures incoming audio into a buffer and lets you slice, rearrange, and transform loops in real time using a grid of modifier pads.

System Requirements

  • macOS 11.0 (Big Sur) or later
  • A VST3-compatible DAW (Ableton Live, Logic Pro, Reaper, etc.)
  • Intel or Apple Silicon Mac

What Gets Installed
